Online Clothing Store

Online clothing store combines cutting-edge technologies such as React, Redux, Node.js, Stripe API, OAuth, and Firebase to offer customers a seamless shopping experience. We've implemented a robust user account creation and login system, including Google login service integration, ensuring a secure and convenient browsing experience. With a focus on user convenience and security, we've incorporated a secure shopping cart, checkout functionality, and credit/debit card payment system powered by Stripe technology. Leveraging unidirectional data flows with Redux and memoized selectors, we've addressed re-render issues and optimized performance, while Redux-Persist ensures smooth state persistence across React components. To enhance responsiveness across various devices, we've utilized CSS media queries and flexbox functionality, ensuring an optimal viewing experience for all users. Hosting on Netlify ensures reliable and scalable performance for our client.
